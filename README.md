## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/utility-server.git
   ```

2. Install dependencies:

   ```bash
   cd utility-server
   npm install
   ```

3. Start the server:

   ```bash
   npm start
   ```
